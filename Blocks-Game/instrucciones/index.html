Instrucciones

Ya sabes como jugar asi que diviertete.

  • Usa las teclas w a s d para mover los bloques.
  • Presiona la barra espaciadora (q o e) para rotar los bloques.
  • Presiona Esc para quitar la musica.
  • Completa líneas para ganar puntos.
