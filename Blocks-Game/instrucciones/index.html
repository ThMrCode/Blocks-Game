Instrucciones

El objetivo del juego es colocar bloques de diferentes formas para crear líneas completas sin dejar espacios. Cuando una línea se completa, desaparece y ganas puntos, los bloques superiores caen como arena perdiendo su forma. El juego termina cuando no hay más espacio disponible para colocar bloques.

  • Usa las teclas de flecha para mover los bloques.
  • Presiona la barra espaciadora para rotar los bloques.
  • Completa líneas para ganar puntos.
